# Sudoku Game in C

A console-based Sudoku game and solver developed using the C programming language.  
This project allows users to play Sudoku by entering numbers and also provides a solution checker.

## Features
- 9×9 Sudoku board display
- User input for filling empty cells
- Validation of Sudoku rules (row, column, and 3×3 grid)
- Simple and interactive console interface
- Written using basic C concepts

## Technologies Used
- C Programming Language
- GCC Compiler (or any standard C compiler)
